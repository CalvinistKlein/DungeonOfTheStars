# FFG RPG Skills Registry: Characteristics & Categories

This file defines the mapping of skills to their base characteristics (Brawn, Agility, Intellect, Cunning, Willpower, Presence) used to calculate dice pools.

---

## 1. General Skills
*   **Athletics**: Brawn (BR)
*   **Computers**: Intellect (INT)
*   **Engineering**: Intellect (INT)
*   **Medicine**: Intellect (INT)
*   **Perception**: Cunning (CUN)
*   **Persuasion**: Presence (PR)
*   **Stealth**: Agility (AG)
*   **Survival**: Cunning (CUN)
*   **Thievery**: Cunning (CUN)

---

## 2. Combat Skills
*   **Blaster Rifles**: Agility (AG)
*   **Heavy Weapons**: Agility (AG)
*   **Melee Weapons**: Brawn (BR)
*   **Ranged Weapons [Light]**: Agility (AG)
*   **Unarmed Combat**: Brawn (BR)

---

## 3. Space & Command Skills
*   **Astrogation**: Intellect (INT)
*   **Piloting [Space]**: Agility (AG)
*   **Tactics [Space]**: Intellect (INT)
*   **Vehicles [Surface]**: Agility (AG)
